# 出力モード

用途に合わせて出力形式を切り替えられます。

## チャート（デフォルト）

何も指定しなければ、カラーのターミナルチャートが表示されます。

```bash
vz data.csv
```

チャートの上に表示されるサマリーラインには:
- チャート種別・軸名
- 値のレンジとスパークライン
- トレンドの方向（↑ +80% のように）
- マルチシリーズの場合は凡例
- 「この列も面白いかも」という提案

## テーブル

数値をざっと確認したいとき。

```bash
vz data.csv -o table
```

棒グラフの場合は集計後の値が、それ以外は元の X/Y データがテーブルとして出力されます。

## スパークライン

たった 1 行の Unicode スパークライン。シェルスクリプトやプロンプトに埋め込むのに便利。

```bash
vz data.csv --spark
# → ▂▅▃▁█▇
```

応用:
```bash
echo "今月の売上: $(vz sales.csv --spark)"
# → 今月の売上: ▂▅▃▁█▇
```

## Info

チャートは要らない、データの概要だけ知りたいとき。

```bash
vz data.csv --info
```

列ごとの型・欠損数・統計量と、vz が推奨するチャート設定が表示されます。

## JSON

プログラムから使いたいとき。メタデータを JSON で出力。

```bash
vz data.csv --info -o json
```

## 3 つのモード

### ワンショット（デフォルト）

チャートを 1 枚描いて即終了。パイプラインの途中や、ちょっと確認したいときに。

### Explore モード

データを対話的に探索する TUI。

```bash
vz explore data.csv
```

| キー | 操作 |
|------|------|
| `h` / `l` | X 軸を変更 |
| `j` / `k` | Y 軸を変更（テーブル時はスクロール）|
| `d` / `Tab` | チャート ↔ テーブル切替 |
| `1`〜`4` | チャート種別を固定 |
| `0` | 自動に戻す |
| `q` | 終了 |

### Present モード

Markdown にチャートを埋め込んで、ターミナル上でスライド発表。

```bash
vz present slides.md
```

こんな Markdown を書く:

````markdown
# 売上レポート

```chart
source: sales.csv
x: month
y: revenue
type: line
title: 月次売上推移
```

---

# まとめ
- 売上は前年比 80% 増
- 東京が全都市をリード
````

操作: `←` / `→` でページ送り、`g` / `G` で先頭/末尾、`q` で終了。
