# 出力モード

vz は用途に応じた複数の出力形式をサポートしています。

## チャート（デフォルト）

カラーターミナルチャートとサマリーラインを描画。

```bash
vz data.csv
```

サマリーラインに含まれる情報:
- チャート種別と軸
- 値域とスパークライン
- トレンド方向 (↑/↓ とパーセンテージ)
- マルチシリーズの凡例
- 追加カラムの提案

## テーブル

構造化テキストテーブル。データの素早い確認に便利。

```bash
vz data.csv -o table
```

棒グラフの場合は集計値、その他は生の X/Y データを表示。

## スパークライン

1行の Unicode スパークライン。スクリプトやダッシュボードへの埋め込みに最適。

```bash
vz data.csv --spark
# 出力: ▂▅▃▁█▇
```

他のツールとの組み合わせ:
```bash
echo "売上推移: $(vz sales.csv --spark)"
# 売上推移: ▂▅▃▁█▇
```

## Info

チャートを描画せずにカラムメタデータを表示。型、統計、推奨チャートを確認。

```bash
vz data.csv --info
```

## JSON

機械可読なメタデータ出力。

```bash
vz data.csv --info -o json
```

## 3つの対話モード

### ワンショット（デフォルト）
チャートを描画して終了。パイプラインやクイック確認に最適。

### Explore モード
vim風ナビゲーションのインタラクティブ TUI。

```bash
vz explore data.csv
```

| キー | 操作 |
|------|------|
| `h`/`l` | X軸変更 |
| `j`/`k` | Y軸変更 / テーブルスクロール |
| `d` / `Tab` | チャート ↔ テーブル切替 |
| `1`-`4` | チャート種別を強制 |
| `0` | 自動（リセット）|
| `q` | 終了 |

### Present モード
ライブチャート埋め込みのターミナルスライド。

```bash
vz present slides.md
```

Markdown にチャートブロックを記述:

````markdown
# 売上レポート

```chart
source: sales.csv
x: month
y: revenue
type: line
title: 月次売上
```

---

# まとめ
- 売上 80% 増
````

操作: `←`/`→` or `h`/`l`、ジャンプ: `g`/`G`、終了: `q`
