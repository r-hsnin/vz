## スパークライン

```bash
$ vz sales.csv --spark
▂▅▃▁█▇

$ vz stock.csv --spark
▁▂▂▃▄▇▇█
```

シェルスクリプトに埋め込める。

```bash
$ echo "売上: $(vz sales.csv --spark)  株価: $(vz stock.csv --spark)"
売上: ▂▅▃▁█▇  株価: ▁▂▂▃▄▇▇█
```

$ vz sales.csv --where "revenue>1500" -o table
