## スパークライン — 1行で傾向を把握

```bash
$ vz sales.csv --spark
▂▅▃▁█▇

$ vz stock.csv --spark
▁▂▂▃▄▇▇█
```

シェルスクリプトに組み込むとこんな使い方も:
```bash
$ echo "売上: $(vz sales.csv --spark)  株価: $(vz stock.csv --spark)"
売上: ▂▅▃▁█▇  株価: ▁▂▂▃▄▇▇█
```

## フィルタリング — 必要な行だけ抽出

```bash
$ vz sales.csv --where "city=Tokyo" -o table
date        revenue
----------  -------
2024-01-01     1000
2024-03-01     1200
2024-05-01     2000

$ vz sales.csv --where "revenue>1500" -o table
date        revenue
----------  -------
2024-05-01     2000
2024-06-01     1800
```
