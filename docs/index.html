Why vz?

Data visualization shouldn't require configuration files, GUIs, or a PhD in charting libraries.

🧠

Smart Auto-Detection

Infers column types (temporal, quantitative, categorical) and automatically selects the best chart type. No flags needed.

⚡

Zero Config

Works out of the box with CSV, TSV, JSON, and NDJSON. Format auto-detected from extension or content.

📊

Multi-Series & Color

Auto-groups data by category with color legends. Supports multi-Y overlay and color grouping with -c.

🔍

Explore Mode

Interactive TUI with vim-style navigation. Switch axes, chart types, and toggle between chart and data table views.

🎬

Present Mode

Build terminal presentations with embedded live charts from Markdown files. Perfect for data-driven talks.

📈

Rich Summary

Every chart includes a smart summary line with sparkline, trend direction, range, and suggestions for additional columns.

Installation

Requires Rust 1.70+. Install with Cargo in seconds.

From Source (recommended)

cargo install --git https://github.com/r-hsnin/vz

Clone & Build

git clone https://github.com/r-hsnin/vz
cd vz && cargo install --path .

Verify Installation

vz --version # vz 0.1.0

Architecture

Clean, modular Rust codebase. Each layer has a single responsibility.

┌─────────────────────────────────────────────────────┐ │ CLI (clap 4) │ │ vz <file> | vz explore <file> | vz present │ └────────────────────────┬────────────────────────────┘ │ ┌────────────────────────▼────────────────────────────┐ │ Data Loader (CSV/TSV/JSON/NDJSON) │ └────────────────────────┬────────────────────────────┘ │ ┌────────────────────────▼────────────────────────────┐ │ Type Inference Engine (auto-detect types) │ └────────────────────────┬────────────────────────────┘ │ ┌────────────────────────▼────────────────────────────┐ │ Chart Selector + Data Builder (shared logic) │ └────────────────────────┬────────────────────────────┘ │ ┌──────────────┼──────────────┐ ▼ ▼ ▼ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ One-shot │ │ Explore │ │ Present │ │ (stdout) │ │ (TUI) │ │ (Slides) │ └──────────────┘ └──────────────┘ └──────────────┘

Quick Reference

Common commands to get you started.

Usage Examples
# Auto-visualize — detects chart type automatically $ vz data.csv # Specify axes $ vz sales.csv -x month -y revenue # Multi-series with color grouping $ vz sales.csv -y revenue -c city # Multiple Y columns overlaid $ vz sales.csv -y revenue,profit # Pipe from stdin $ cat data.json | vz # Interactive exploration $ vz explore data.csv # Slide presentation $ vz present slides.md # Filter + table output $ vz data.csv --where "sales>100" -o table # Shell completion $ vz completions bash >> ~/.bashrc
